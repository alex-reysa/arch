# Implementation plan

The full V1 implementation roadmap lives under `founding-docs/` in
[`../founding-docs/IMPLEMENTATION_PLAN.md`](../founding-docs/IMPLEMENTATION_PLAN.md).

This `docs/` tree holds derived documentation that is meant to evolve as the
code ships:

- [`decisions/`](./decisions) — Architecture Decision Records covering choices
  the plan flagged as load-bearing.

The founding document remains the canonical V1 plan; ADRs override or refine
it only when a decision changes after a section has already been written.
